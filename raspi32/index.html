  • Raspberry Pi cluster

    This website presents the Bachelor project initiated by Sidney Bovet at the Swiss Federal Institute of Technology in Lausanne (EPFL) in september 2013. The project aimed to build a low-cost educational cluster that can be used in a Distributed Storage course for instance or for research purpose. It features a self-evaluation mechanism, node activity measurement and implements the famous Message Passing Interface as a cluster-computing interface.

    The semester project is now finished, but the cluster itself still has room for improvement. While not being a Work In Progress, this project might still be changing over time.
